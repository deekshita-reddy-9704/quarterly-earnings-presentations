Sample Python Calculation


# Calculate CAGR
revenue_start = 5_000_000
revenue_end = 10_000_000
years = 3
cagr = ((revenue_end / revenue_start)**(1/years)) - 1
print(f"CAGR: {cagr:.2%}")

Financial Formula

Net Present Value (NPV):

$$NPV = \sum_{t=0}^{T} \frac{R_t}{(1+i)^t}$$
